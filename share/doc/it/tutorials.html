Tutorials

Real world experience helps a lot when it comes to using any software. Please visit the Scribus Wiki for more tutorials on how to perform specific tasks with Scribus. Among others, a great "Getting Started with Scribus" tutorial, orginally created by Niyam Bhushan and subsequently enhanced and translated into other languages by others, is available there. It covers basic layout theory, workflow and describes how to apply it to the creation of your next newsletter. It's not just about words and pictures on paper! Read this and find out why. You can directly access the tutorial from within Scribus via Help > Tutorials.
